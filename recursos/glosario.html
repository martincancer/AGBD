Glosario

Definiciones simples

Base de datos relacional
Conjunto de información perteneciente a un mismo contexto organizado en tablas (filas y columnas) que se conectan entre sí mediante identificadores únicos (claves)
Tabla
Unidad fundamental de almacenamiento. Se organiza de manera bidimensional en filas y columnas, conteniendo datos que pertenecen a una entidad específica.
Clave primaria (PK)
Columna (o conjunto de columnas) que identifica de forma única cada fila de una tabla.
Clave foránea (FK)
Columna que referencia la clave primaria de otra tabla, materializando una relación.
Normalización
La normalización de bases de datos es el proceso de organizar la información en tablas más pequeñas y lógicas para evitar la duplicación de datos y prevenir errores al actualizarlos
SQL
SQL (Lenguaje de Consulta Estructurada / Structured Query Languaje) es el lenguaje de programación estándar utilizado para comunicarse con bases de datos relacionales
Dato
Un dato es la unidad mínima de información (un número, una palabra o una fecha) que, al agruparse y relacionarse con otros, permite tomar decisiones o crear conocimiento útil.
Relación uno a muchos (1:N)
Vínculo en el que un registro de una tabla puede asociarse a muchos registros de otra, pero cada uno de estos pertenece a un solo registro "padre".
